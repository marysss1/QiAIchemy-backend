# 中医基础理论与诊断学青年亚健康导向摘要（约3000字）

## 主要来源（教材与规范线索）
- 《中医基础理论》通行教学框架：阴阳五行、脏腑、气血津液、病因病机
- 《中医诊断学》通行框架：四诊、八纲、脏腑辨证、气血津液辨证
- 说明：本文为结构化学习摘要，用于健康管理场景的可解释输出。

## 一、青年亚健康的基本认识：功能失衡先于器质性病变
中医基础理论强调“整体观”和“恒动观”。青年亚健康常见的不是单器官重病，而是跨系统的功能失衡：睡眠节律紊乱、情绪应激持续、饮食结构失衡、活动不足与久坐叠加，导致“气机失调、脏腑失和、气血津液运行失衡”。这与现代“慢性低度应激负荷”概念可相互印证。

判断青年亚健康不能只看一两项主诉，应把“体感+行为+指标”放到同一张图里：体感包括疲劳、焦虑、脑雾、胃口波动；行为包括作息、饮食、运动、屏幕暴露；指标包括睡眠时长、HRV、静息心率、活动量、血糖等。

## 二、核心理论在实践中的映射
### 1）阴阳
阴阳不是抽象名词，而是“兴奋-恢复、外放-内守、消耗-修复”的动态平衡。青年常见“阳不入阴”：白天体力脑力透支，夜间却难以入睡。也常见“阴不敛阳”：夜间多梦易醒、晨起不解乏。

### 2）气血津液
- 气：推动与调节，气不足则乏力、懒言、恢复慢；
- 血：濡养与承载，血不足可见心悸、头晕、注意力下降；
- 津液：滋润与代谢，津亏见口干便结，津滞则痰湿内生。

青年亚健康常呈“气机不畅+津液失调”并见：压力大时气郁，久坐和饮食失衡时痰湿，熬夜和刺激过多时阴津暗耗。

### 3）脏腑协同
- 心主神明：关系睡眠、专注与情绪稳定；
- 肝主疏泄：关系压力应对、情绪调节、节律弹性；
- 脾主运化：关系食欲、消化、体成分管理；
- 肺主气：关系呼吸节律与活动耐力；
- 肾藏精：关系长期恢复储备。

青年亚健康往往不是“单脏问题”，而是“肝脾心”或“心肾”联动。

## 三、诊断学方法：四诊合参与八纲归纳
### 1）四诊合参
- 望：神色、体态、舌象、精神状态；
- 闻：语音气息、呼吸状态；
- 问：睡眠、饮食、二便、情绪、经期、工作负荷；
- 切：脉象与局部触诊（临床场景）。

在数字健康场景中，问诊与指标追踪尤为关键，可先形成“远程初筛”，再决定是否线下就医。

### 2）八纲辨证
- 表里：问题偏外在/偏内在；
- 寒热：偏寒象/偏热象；
- 虚实：正气不足/邪实壅滞；
- 阴阳：总体属性归纳。

青年人常见“虚实夹杂”：既有疲劳（虚），又有郁滞与热象（实）。

## 四、常见辨证模式（青年高频）
1. 肝郁气滞倾向：压力大、情绪波动、胸胁不舒、睡眠浅；
2. 脾虚夹湿倾向：餐后困、腹胀、体重上行、注意力涣散；
3. 心神不宁倾向：入睡困难、梦多易醒、次日疲劳；
4. 气阴两虚样消耗：长期透支后精力下降、口干、恢复慢；
5. 痰湿内阻倾向：久坐少动、头重困倦、体能下降。

## 五、与 HealthKit 的结构化对接建议
可构建“指标-证候倾向”映射，但必须加不确定性说明：
- 睡眠评分/时长下降 -> 心神不宁、阴津不足风险上升；
- HRV下降 + 静息心率升高 -> 应激负荷增大、恢复不足；
- 步数长期不足 -> 气机运行不畅、阳气不展；
- 血糖异常波动 -> 脾胃代谢与生活节律失衡，需优先医学评估。

原则：单点数据不下结论，趋势数据用于提示与复盘。

## 六、青年亚健康管理流程（可执行）
1. 识别阶段（第1周）
- 记录睡眠、活动、情绪、胃肠状态；
- 明确最大影响因子（作息、压力、饮食、久坐）。

2. 干预阶段（第2-4周）
- 固定作息窗口；
- 饭后步行与每周有氧+抗阻；
- 降低晚间刺激，减少夜宵与高糖饮料。

3. 巩固阶段（第5周后）
- 建立高压周预案；
- 以周为单位复盘指标与体感一致性；
- 出现红旗信号即转诊。

## 七、沟通与输出规范（用于Agent）
- 先给“风险分层”，再给“可执行建议”，最后给“复盘指标”；
- 建议分三层：当天可做、7天计划、需就医事项；
- 保持审慎措辞：使用“倾向、可能、建议进一步评估”，避免绝对化诊断。

## 八、边界与伦理
中医理论可提供解释框架，但不能替代现代医学检查。对血氧、血糖、血压、突发胸痛、意识异常等高风险情况，应明确提示线下医疗优先。数字化中医建议的价值在于“早识别、早调整、少走弯路”，而非替代诊疗。

## 九、四诊信息结构化（面向数据建模）
为了让后端检索更稳定，可把四诊内容标准化为字段：  
- 望诊字段：精神状态、面色、体态变化、舌色舌苔；  
- 闻诊字段：语速语调、呼吸深浅、口气变化；  
- 问诊字段：睡眠、食欲、排便、月经、压力源、运动、咖啡因/酒精；  
- 切诊字段（若有线下结果）：脉象关键词与医师结论。  
这样可在知识库中做“证据锚点”，避免生成只基于单一症状的建议。

## 十、八纲与脏腑辨证的组合规则
青年亚健康常见不是单一八纲结论，而是“主证 + 兼证”：  
1. 主轴先定虚实：是以虚为主（恢复不足）还是以实为主（郁滞壅塞）；  
2. 次轴再定寒热：是偏寒（功能低下）还是偏热（应激亢奋）；  
3. 最后落脏腑：肝（情志与疏泄）、脾（运化与代谢）、心（睡眠与神志）三者最常联动。  
例如“虚实夹杂、偏热、肝脾不和”在青年人中很常见，对应策略就应是“先降应激和晚间刺激，再稳脾胃，再谈训练进阶”。

## 十一、舌象与体感结合的审慎使用
数字化场景中常希望用舌象快速判断，但应注意：  
- 舌象受拍摄光线、角度、饮食、短时状态影响较大；  
- 不宜用一次舌照下确定诊断，应结合连续体感和行为数据；  
- 更适合用于“趋势观察”，如苔厚腻反复出现是否与作息饮食失衡同步。  
系统输出应使用“倾向于、可能提示”，并提示用户必要时面诊复核。

## 十二、面向青年亚健康的最小诊断闭环
推荐每次对话至少完成以下闭环：  
1. 当前最困扰症状（1-2个）；  
2. 近7天行为偏移（睡眠、饮食、活动）；  
3. 近7天指标偏移（HRV、静息心率、步数、睡眠）；  
4. 证候主轴归纳（不超过2条）；  
5. 下周可执行动作（不超过3条）；  
6. 红旗风险提醒（是否需要就医）。  
该闭环可显著降低“建议很多但无法执行”的问题。

## 十三、证候与行为因子的双向映射
为了避免“只讲理论不讲执行”，建议建立证候-行为双向表：  
- 肝郁气滞倾向 <- 高压、久坐、情绪抑制、长期晚睡；  
- 脾虚夹湿倾向 <- 进食不定时、夜宵、久坐、活动不足；  
- 心神不宁倾向 <- 夜间信息刺激、焦虑反刍、咖啡因过晚；  
- 气阴耗伤倾向 <- 长期超负荷输出、恢复时间不足。  
反过来，每个证候都要绑定最小行动单元，例如“固定起床时间、每日户外光照、餐后步行、晚间降刺激”。

## 十四、青年亚健康问卷条目建议（可直接入库）
可设计一个20-30题的快速问卷，核心覆盖：  
1. 睡眠：入睡时间、夜醒次数、晨起恢复感；  
2. 情绪：焦虑、烦躁、低落频率；  
3. 饮食：夜宵频率、甜饮摄入、饭后困倦；  
4. 活动：步数、运动频率、久坐时长；  
5. 消化：食欲、腹胀、排便规律；  
6. 女性专项：经前情绪、痛经、经期睡眠。  
问卷结果用于“初筛与趋势追踪”，不作为确诊工具，系统需明确提示这一点。

## 十五、辨证输出的语言规范
为避免医疗风险，建议统一措辞规范：  
- 用“倾向于、提示、建议评估”替代“就是、确定是”；  
- 明确数据来源（用户主诉/设备指标/既往史）；  
- 区分“日常调理建议”与“就医指征”；  
- 对高风险症状使用强提醒语句。  
这种规范可提升系统可信度，也便于后期伦理审查和论文答辩。

## 十六、知识库继续扩容时的优先顺序
从工程效率看，建议优先扩充三类资料：  
1. 中医基础理论与诊断学教材中的辨证逻辑章节；  
2. 青年高频问题相关临床路径（睡眠、焦虑、脾胃、代谢）；  
3. 指南型资料（运动、营养、心理健康）与中医框架对照内容。  
扩容后可按“理论-症状-行为-指标-建议”五段式切块，检索质量会明显优于纯长文本堆叠。

## 十七、可用于建模的诊断字段细表（补充）
为支持后端一致化推理，建议把每次对话转成标准字段：  
- 基本信息：年龄段、性别、工作作息、是否倒班；  
- 主诉字段：疲劳、睡眠、情绪、消化、疼痛等主症评分；  
- 诱因字段：熬夜、饮食失节、久坐、情绪事件、经期；  
- 体感字段：寒热偏向、口渴口黏、汗出、二便；  
- 数据字段：睡眠时长、HRV、静息心率、步数、血糖血氧；  
- 风险字段：胸痛、晕厥、严重呼吸困难、明显精神危机。  
有了字段化输入，模型可以先做规则层分流，再做生成层解释，稳定性更高。

## 十八、青年亚健康辨证最小词表（建议入图谱）
建议维护一个可扩展词表：  
- 情绪类：焦虑、易怒、压抑、反刍、社交回避；  
- 睡眠类：入睡难、夜醒、多梦、早醒、晨起疲惫；  
- 消化类：纳呆、腹胀、反酸、便溏、便秘；  
- 代谢类：体重上行、餐后困、口渴、乏力；  
- 体能类：气短、恢复慢、肌肉酸痛迁延。  
词表与证候节点绑定后，可作为 Graph-RAG Lite 的节点种子，减少检索偏题。

## 十九、诊断到干预的闭环示例（工程化）
当系统识别到“肝郁气滞夹脾虚”时，输出应包含：  
1. 解释：为何会出现胸闷、焦虑、餐后困；  
2. 行动：固定起床、晚间减刺激、餐后步行、减少甜饮；  
3. 复盘：7天后对比睡眠、HRV、步数与体感；  
4. 升级：若胸闷持续加重或出现红旗，立即就医。  
这种模板可直接作为后端响应协议，提高前后端协作效率。

## 二十、补充说明
基础理论与诊断学在工程中的价值，是把“经验”变成“结构化流程”。当流程被固定后，模型能力波动对结果的影响会明显下降。对本科生项目而言，这种稳定可复现的流程化设计，比单次回答的文采更重要。

## 二十一、扩展补记
建议后续再加入“体质辨识结果与行为处方映射表”，并用统一字段写入数据库。这样无论是传统 RAG 还是图增强检索，都能直接读取结构化特征，提高系统一致性与可解释性。
