# Knowledge Sources (No Fabrication Rule)

Put real source texts in these folders before running `npm run knowledge:build`.

## Folder mapping
- `01_huangdi_neijing` -> 黄帝内经语料
- `02_shanghan_jingui` -> 伤寒论 / 金匮要略语料
- `03_wenbing_tiaobian` -> 温病条辨语料
- `04_foundation_diagnosis` -> 中医基础理论 / 中医诊断学教材语料
- `05_chronic_youth_subhealth` -> 青年亚健康与慢病管理指南语料

## Allowed file types
- `.txt`, `.md`, `.markdown`

## Important
- Do not paste AI-generated content as source text.
- Keep source provenance (book title / chapter / publisher / year) in file names or file headers.
- Ensure you have rights to use the source for your project.
